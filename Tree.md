优点：计算复杂度不高，输出结果易于理解，对中间值的缺失不敏感，可以处理不相关特征数据。
缺点：可能会产生过拟合
伪代码：
If so return 类标签；
Else
    寻找划分数据集的最好特征
    划分数据集
    创建分支节点
        for 每个划分的子集
            调用函数createBranch并增加返回结果到分支节点中
    return 分支节点
划分标准：信息增益
	信息：l(xi) = -log(p(xi), 2)
	熵：H = -(p(x1)log(p(x1), 2)+...+p(x2)log(p(x2), 2))
熵最大的特征作为划分标准
